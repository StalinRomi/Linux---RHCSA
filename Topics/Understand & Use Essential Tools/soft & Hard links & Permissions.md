Hard Link ->

**ln** file name

Soft link ->

**ln -s** file name

Permissions ->

1. **Absolute** method

**chmod 754** file name
( Here, 7 -> For User, 5 -> For others, 4 -> For others)

2. **Symbolic** method

**chmod u+rwx** file name
(Here, we've give read, write & execute permission to user)

Similarly, we can delete the permissions as well
**chmod u-rwx** file name
